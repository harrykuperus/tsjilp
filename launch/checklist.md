# Launch Checklist & Schedule

## Before you launch

- [ ] Code cleaned and copied into the repo structure
- [ ] `.env.example` matches what the app actually needs
- [ ] `docker compose up` works from a fresh clone
- [ ] Database schema in `database/schema.sql`
- [ ] No secrets, no IMP business code, no real user data
- [ ] GitHub repo created at github.com/harrykuperus/tsjilp
- [ ] Push to main
- [ ] Add GitHub topics/tags: `chat`, `ai`, `privacy`, `self-hosted`, `php`, `openai`, `communication`, `agpl`

## Launch order (timing matters)

### Day 1 — Tuesday or Wednesday (best HN days)

1. **Hacker News** — Morning US time (8-10am ET / 2-4pm CET)
   - Post: `launch/hackernews.md`
   - Title must start with "Show HN:"
   - Be in the comments for the first 2 hours answering questions

2. **r/selfhosted** — Same day, slightly after HN
   - Post: `launch/reddit-selfhosted.md`
   - Engage in comments

3. **r/privacy** — Same day, after selfhosted
   - Post: `launch/reddit-privacy.md`
   - Privacy angle is the hook here, not the tech

### Day 2-3 — Follow up

4. **Dev.to article** — Publish the long-form piece
   - Post: `launch/dev-to-article.md`
   - Cross-post to Hashnode if you have an account
   - Share the article link on Reddit and HN comments if relevant

### Week 1+ — Ongoing

5. **Product Hunt** — Schedule a launch day
   - Open-source products do well on PH
   - Need a good tagline: "Chat app with a private AI secretary"
   - Prepare screenshots/GIF before launching

6. **Lobste.rs** — If HN goes well
   - Smaller, more technical audience
   - Tag with `privacy`, `ai`

## Tips

- **Don't post all platforms at the same time.** You need to be present for comments.
- **HN is the biggest single traffic source.** Prioritize being available for the first 2 hours.
- **Screenshot or GIF of the assistant in action** is worth 1000 words. Add one to the README before launching.
- **Reddit has rules about self-promotion.** Post naturally, engage genuinely, don't just drop links.
- **The "assistant as secretary" metaphor is your strongest differentiator.** Lead with it everywhere.
